Catálogo de productos. Dos roles definidos

El usuario base puede consultar los productos asociados con su id de usuario.

El administrador puede consultar todos los productos
El administrador puede actualizar, crear y borrar productos.

Hay una pantalla de login
Tras un login exitoso, se accede a la lista de productos. El usuario base solo puede consultar esta vista. El administrador puede ver botones para crear, editar y borrar productos.
